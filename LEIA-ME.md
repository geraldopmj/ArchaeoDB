# ArchaeoDB

O ArchaeoDB é um software desenvolvido para facilitar o gerenciamento e análise de dados em zooarqueologia. Ele permite a catalogação, visualização e manipulação de informações sobre vestígios ósseos, auxiliando pesquisadores na organização e interpretação de seus dados.

## Principais Recursos:

*   **Gerenciamento de Dados**: 
Gestão completa de Sítios, Coleções, Amostras, Unidades de Escavação, Níveis, Materiais e Espécimes.
*   **Banco de Dados**: Utiliza SQLite para armazenamento de dados robusto e portátil. Crie novos bancos de dados ou abra existentes facilmente.
*   **Importação e Exportação**:
    *   Crie e atualize bancos de dados diretamente de arquivos Excel.
    *   Exporte o banco de dados completo para Excel.
    *   Gere relatórios em PDF para Materiais, Unidades e Espécimes.
*   **Estatísticas e Visualização**:
    *   Gere gráficos para Tipos de Materiais, Descrições e Quantidades.
    *   Visualize a densidade por Unidade.
    *   Calcule e plote NISP (Número de Espécimes Identificados) por Táxon.
    *   Exporte gráficos para PNG.
*   **Experiência do Usuário**:
    *   Interface intuitiva com filtragem hierárquica.
    *   **Suporte multilíngue**: Inglês, Português e Espanhol.
    *   **Temas**: Modos Claro e Escuro.
